# Rachel Levine-Katz Counselling

A website for Rachel Levine-Katz, Registered Social Worker offering grief and bereavement counselling in Ottawa, ON.

## Structure

- **index.html** — Home page with hero and intro
- **about.html** — About Rachel and her approach
- **services.html** — Areas of support and what to expect
- **pricing.html** — Session fees, payment methods, and insurance
- **contact.html** — Contact information

## Running locally

Open `index.html` in a browser, or use a simple local server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000

## Customization

- **Email**: Update the email address in `contact.html` if needed (currently set to rachel@rachellevinekatz.com).
- **Assets**: Logo is in `assets/logo.png`, profile photo in `assets/rachel-profile.png`.
- **Colours**: Defined in `styles.css` using the brand palette from the style guide.
